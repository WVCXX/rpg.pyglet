# The Life and Suffering of Prince Jerian

Текстовая RPG в мрачном средневековом сеттинге с глубоким сюжетом, системой выбора и последствий.

## Особенности

- 🎮 **8+ локаций** с уникальными описаниями
- 👥 **50+ NPC** с характерами и расписанием
- ⚔️ **Боевая система** с врагами и боссами
- 💰 **Экономика** с меняющимися ценами
- 🔮 **Магия** с кругами и школами
- ❤️ **Романтика** и отношения
- 🏥 **Болезни и травмы**
- 💀 **Несчастные случаи**
- 📜 **Квесты** с ветвлением
- 🎨 **Красивый GUI** на tkinter

## Установка

```bash
# Клонирование
git clone https://github.com/WVCXX/prince_jerian_rpg.git
cd prince_jerian_rpg

# Установка зависимостей
pip install -r requirements.txt

# Запуск
python run.py
